ajaxType

ajaxType: "fetch"

Inherited from Request.ajaxType

  • Defined in src/index.type.ts:221

"xhr" or "fetch".

When a request is generated by XMLHttpRequest, then ajaxType is "xhr".

When a request is generated by window.fetch, then ajaxType is "fetch".
